Instruction of using visual_grating.ino/visual_grating.py
*upload and run the arduino code first and then run the python code
*then open the serial monitor of Arduino
*(arduino) ask for a char type input from serial monitor and store this input as "command"
*close the arduino serial monitor and the command will be send to python
*(python) if command is A, then trigger visual grating with orientation of 45 degree
*(python) if command is B, then trigger visual grating with orientation of 135 degree
